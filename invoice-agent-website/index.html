核心功能

覆盖从发票获取、AI识别到报销表生成的完整流程

📄

PDF 发票识别

支持上传 PDF 发票，自动解析发票内容并提取关键字段。

📬

邮箱自动读取

支持 QQ / 163 / 126 邮箱，通过 IMAP 授权自动获取发票邮件。

🤖

AI 结构化提取

调用通义千问 / Dify / Ollama，提取发票号码、金额、日期、销售方等信息。

📊

Excel 报销表

识别完成后自动生成 Excel 报销表，方便下载、提交和归档。

使用流程

01

上传或读取发票

用户可以上传 PDF，也可以从邮箱自动读取发票附件。

02

AI 自动识别

系统调用大模型进行字段提取，并转换为结构化数据。

03

生成报销表

自动整理识别结果，生成可下载的 Excel 报销表。

这个项目不仅完成了功能开发，还包含测试验证、异常分析、Token 成本控制和结果报告， 更接近真实 AI 产品从 0 到 1 的开发过程。
